# 👁️ Lash Designer - Website

Landing page profissional para estúdio de extensão de cílios, focada em apresentação de serviços e agendamentos.


## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript

## ✨ Funcionalidades
- Catálogo de serviços (Fio a fio, Volume Russo, etc).
- Galeria de fotos dos resultados.
- Botão direto para agendamento via WhatsApp.
- Design responsivo para celulares.
